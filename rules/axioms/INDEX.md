# Axioms 索引

> **公理（Axioms）** 是你的核心认知滤镜，从个人经历中提炼的深层决策原则。
> 
> 不同于 Skills（执行层，"怎么做"），Axioms 是思考层（"为什么"），用于启发深度思考。

---

### 什么样的观察值得晋升为公理

好的 axiom 必须满足：
- **持久性**：跨时间反复出现
- **独特性**：属于你个人的，不是通用常识
- **预测性**：能预测你未来的决策倾向
- **边界性**：有明确的不适用条件

### Axiom 文件格式（最简版）

```markdown
# 公理名称

## 核心公理
一段话，自含的核心陈述。

## 证据
- 来自 observations 的具体事例 1
- 来自 observations 的具体事例 2
- 来自 observations 的具体事例 3

## 边界条件
什么情况下这个公理不适用。
```

深度版本可以随时间扩展为：核心公理 / 深度推演 / 适用判断表 / 陷阱 / 关联公理 / 实践建议。
