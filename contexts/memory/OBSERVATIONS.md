# Observations（观察记录）

> 三层记忆系统的 L1/L2 层。每日观察在这里积累，每周反思在这里蒸馏。
>
> 红黄绿优先级系统：
> - 🔴 红：跨项目的方法论/约束洞察，3 个月后仍有复用价值
> - 🟡 黄：活跃项目的决策/进展，未来几周内需要
> - 🟢 绿：常规任务完成记录，1-2 周后可清理
